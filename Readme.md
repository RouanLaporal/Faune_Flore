# Faune_Flore

Faune_Flore is a class [project](http://rouanito.pythonanywhere.com/) developped in python with the framework Django during my third year in French Guiana University for two biologist from the [CNRS](https://www.cnrs.fr/fr/page-daccueil)

## Installation
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Django.

```bash
pip install django
```
